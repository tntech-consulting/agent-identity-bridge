## What is an AIB Passport?

Each passport is a cryptographic identity for an AI agent:
- **passport_id**: Unique URN (`urn:aib:agent:<org>:<slug>`)
- **Ed25519 signature**: Tamper-proof proof of creation
- **Protocols**: Communication standards the agent speaks (MCP, A2A, AG-UI)
- **Tier**: `permanent` / `session` / `ephemeral`

Every agent operation generates a signed **receipt** stored on-chain for audit trail.
