Create, list, revoke, translate, track. Every call is authenticated, metered, and audited.

Créer, lister, révoquer, traduire, suivre. Chaque appel est authentifié, comptabilisé et audité.

POST

passport-create

Create an Agent Passport with protocol bindings, capabilities, and TTL. Auto-generates URN, stores in PostgreSQL, emits audit receipt.

GET

passport-list

List all passports for your org. Filter by status (active/revoked/expired). Paginated. Sorted by creation date.

POST

passport-revoke

Permanently revoke a passport. Cascade-revokes all child passports in the delegation chain automatically.

POST

translate

Translate credentials between A2A Agent Cards, MCP Server Cards, and AG-UI Descriptors. Sub-millisecond latency.

GET

usage-check

Real-time usage: transactions, passports, translations, webhooks. Per-org quotas and limits.

GET

usage-history

Daily analytics for the last 90 days. Action breakdown, latency, error rates. Chart.js-ready format.

POST

policy-manage

CRUD for 12 policy rule types. Deliverable gates, separation of duties, rate limits, time restrictions.

POST

webhook-manage

Register webhook endpoints. HMAC-SHA256 signed. 4 event types: passport.created, revoked, policy.violation, translate.completed.

POST

auth

Signup, login, generate API keys. Auto-provisions org, first key, and beta_pro plan.
