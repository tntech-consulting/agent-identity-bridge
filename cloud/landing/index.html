ENDPOINTS

9 endpoints. Full lifecycle.

Create, translate, enforce, audit — everything your agents need.

POST
passport-create
Create passport with protocol bindings, capabilities, TTL. Ed25519 signed receipt.
GET
passport-list
List passports. Filter by status, paginated, sorted by date.
POST
passport-revoke
Revoke + cascade children. Separation of duties enforced.
POST
translate
A2A ↔ MCP ↔ AG-UI. 6 paths. Sub-millisecond. Webhook fired.
GET
usage-check
Real-time quotas: transactions, passports, translations, webhooks.
GET
usage-history
Daily analytics for 90 days. Chart.js-ready format.
POST
policy-manage
CRUD 12 rule types. Deliverable gates, rate limits, time restrict.
POST
webhook-manage
Register endpoints. HMAC-SHA256 signed. 4 event types.
POST
auth
Signup, login, API key generation. Auto-provisions org + beta_pro.
