FRAMEWORKS

3 lines. Any framework.

3 lignes. N'importe quel framework.

Integrate AIB with LangChain, CrewAI, OpenAI Agents, AutoGen, DSPy, and LlamaIndex.

Intégrez AIB avec LangChain, CrewAI, OpenAI Agents, AutoGen, DSPy et LlamaIndex.

🦜 LangChain
Add AIB identity to any LangChain agent with the AIBTool wrapper.
Ajoutez l'identité AIB à n'importe quel agent LangChain.
from aib import AIBTool
# Wrap your LangChain agent
tool = AIBTool("booking")
agent.tools.append(tool)
🤖 CrewAI
Give every CrewAI agent a portable passport.
Donnez à chaque agent CrewAI un passeport portable.
from aib import CrewAIBridge
# Assign passport to crew
bridge = CrewAIBridge("support")
crew.add_identity(bridge)
🧠 OpenAI Agents
Connect OpenAI Agents SDK for cross-protocol identity.
Connectez le SDK OpenAI Agents pour l'identité cross-protocole.
from aib import OpenAIBridge
# Bridge to MCP/A2A
bridge = OpenAIBridge("analyst")
agent.register(bridge)
🔄 AutoGen
Microsoft AutoGen agents with AIB portable passports.
Agents Microsoft AutoGen avec des passeports AIB portables.
from aib import AutoGenBridge
# Multi-agent conversations
bridge = AutoGenBridge("coder")
agent.set_identity(bridge)
📊 DSPy
Add identity to DSPy pipelines for authenticated inference.
Ajoutez l'identité aux pipelines DSPy pour l'inférence authentifiée.
from aib import DSPyBridge
# Authenticated pipeline
bridge = DSPyBridge("pipeline")
module.passport = bridge
🦙 LlamaIndex
LlamaIndex agents with cross-protocol identity and audit.
Agents LlamaIndex avec identité cross-protocole et audit.
from aib import LlamaIndexBridge
# RAG agent with passport
bridge = LlamaIndexBridge("rag")
agent.identity = bridge

Install the SDK

Installez le SDK

$ pip install agent-identity-bridge
