FRAMEWORKS

3 lines. Any framework.

3 lignes. N'importe quel framework.

AIB provides native integrations for LangChain, LangGraph, CrewAI, OpenAI Agents SDK, Google ADK, Microsoft Agent Framework, and OpenClaw. Python SDK for cloud frameworks, CLI skill for self-hosted agents.

AIB fournit des intégrations natives pour LangChain, LangGraph, CrewAI, OpenAI Agents SDK, Google ADK, Microsoft Agent Framework et OpenClaw. SDK Python pour les frameworks cloud, skill CLI pour les agents self-hosted.

🦜 LangChain
Returns 6 @tool-decorated functions compatible with any LangChain agent (ReAct, OpenAI Functions, etc.).
from aib.integrations import get_langchain_tools

tools = get_langchain_tools(secret_key="your-key")

# Use with any LangChain agent
from langchain.agents import create_react_agent
agent = create_react_agent(llm, tools, prompt)
🤖 CrewAI
Returns 6 BaseTool subclasses with Pydantic input schemas, ready for CrewAI agents.
from aib.integrations import get_crewai_tools

tools = get_crewai_tools(secret_key="your-key")

# Assign to a CrewAI agent
from crewai import Agent
agent = Agent(role="Identity Manager", tools=tools)
🧠 OpenAI Agents SDK
Returns 6 @function_tool functions compatible with the OpenAI Agents SDK.
from aib.integrations import get_openai_agents_tools

tools = get_openai_agents_tools(secret_key="your-key")

# Create agent with AIB identity tools
from agents import Agent
agent = Agent(name="Identity Manager", tools=tools)
🦞 OpenClaw 188K+ ★
AgentSkills-compatible skill. Pre-action guardrails, Ed25519 identity, EU AI Act governance, and cross-protocol translation — enforced before every tool call.
# Install the AIB skill
$ clawhub install aib-identity-governance

# Or manually
$ bash ~/.openclaw/skills/aib-identity-governance/scripts/setup.sh

# Guard runs before every tool call
# Exit 0 = ALLOW, Exit 1 = DENY
$ aib guard check --action "exec.run" \
  --params '{"command":"ls"}'
🕸️ LangGraph 28K ★
Stateful multi-agent workflows as directed graphs. Used in production by Klarna, Replit, and Elastic. AIB passport verification and policy enforcement at every node — with checkpointing and human-in-the-loop support.
from aib.integrations import get_langgraph_tools
from langgraph.graph import StateGraph

tools = get_langgraph_tools(secret_key="your-key")

# Bind AIB tools to any LangGraph node
graph = StateGraph(AgentState)
graph.add_node("identity", tools.passport_node)
🔷 Google ADK 15K ★ · A2A
Google Agent Development Kit — model-agnostic, deployment-agnostic, with native A2A protocol support. AIB translates credentials between A2A, MCP, and AG-UI — making cross-framework agent communication seamless.
from aib.integrations import get_google_adk_tools
from google.adk import Agent

tools = get_google_adk_tools(secret_key="your-key")

# A2A ↔ MCP ↔ AG-UI translation built-in
agent = Agent(name="identity-agent", tools=tools)
agent.run()
🪟 Microsoft Agent Framework Public Preview
Successor to AutoGen + Semantic Kernel. Graph workflows, MCP + A2A support, and human-in-the-loop. AIB adds Ed25519 passport signing and EU AI Act audit trails for enterprise compliance.
from aib.integrations import get_msaf_tools
# pip install agent-framework
from microsoft.agents import AssistantAgent

tools = get_msaf_tools(secret_key="your-key")

# AIB audit trail on every agent exchange
agent = AssistantAgent("identity-manager",
  tools=tools)

Direct usage without frameworks

# Use AIBToolkit directly (no framework dependency)
from aib.integrations import AIBToolkit

tk = AIBToolkit(secret_key="your-key")
result = tk.create_passport("myorg", "booking", "mcp,a2a")
print(result)

# Translate credentials
translated = tk.translate_credential(
  source_json, "a2a_agent_card", "mcp_server_card"
)
