Deploy a kit in 3 steps

Déployez un kit en 3 étapes

1

Choose a kit

Select a pre-built compliance kit or create your own.

Choisissez un kit

Sélectionnez un kit pré-configuré ou créez le vôtre.

2

Deploy via API

One POST to /policy-manage with the kit's JSON payload.

Déployez via API

Un POST à /policy-manage avec le payload JSON du kit.

3

Enforced instantly

All rules are evaluated on every action. Ed25519 signed receipts.

Appliqué instantanément

Toutes les règles sont évaluées à chaque action. Reçus signés Ed25519.

# Deploy the EU AI Act Compliance Kit
curl -X POST /functions/v1/policy-manage \
  -H "x-api-key: aib_sk_live_..." \
  -d '{"kit": "eu-ai-act", "severity": "block"}'

✓ 5 rules deployed · severity: block · active immediately
