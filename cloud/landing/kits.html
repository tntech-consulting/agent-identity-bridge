COMPLIANCE KITS

One protocol. Every regulation.

Industry-specific compliance kits built on the AIB protocol. Each kit adds sector-specific fields, audit trails, and policy templates to your agent identity infrastructure.

Un protocole. Chaque réglementation.

Kits de conformité sectoriels construits sur le protocole AIB. Chaque kit ajoute des champs, audit trails et templates de règles spécifiques à votre infrastructure d'identité agent.

Choose a template

Deploy with your API key

Déployez avec votre clé API

Get a free API key from the Dashboard. Rules are enforced instantly — Ed25519 signed receipts on every action.

Obtenez une clé API gratuite depuis le Dashboard. Les règles s'appliquent immédiatement — reçus signés Ed25519 à chaque action.

# Real API call — generated by the Kit Builder below
curl -X POST https://vempwtzknixfnvysmiwo.supabase.co/functions/v1/passport-create
  -H "x-api-key: aib_sk_live_..."
  -d '{"name":"my-agent","template":"compliance-agent"}'

curl -X POST https://vempwtzknixfnvysmiwo.supabase.co/functions/v1/policy-manage
  -H "x-api-key: aib_sk_live_..."
  -d '{"action":"create","passport_id":"<id>","rule_type":"rate_limit","config":{"max_per_hour":100},"severity":"block"}'
